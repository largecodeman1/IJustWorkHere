SSH vs VNC. Whats the difference?

By Nathaniel Cherian

What is SSH?

The SSH (secure shell) protocol allows you to remotely connect to a computer and use it's interface. Connections can happen from either a local network or across the internet. In the latter it is advised to set up a private/public key pair to prevent unauthorized users from entering your server. The main difference between a VNC and SSH is that VNC provides a graphical desktop that you can use with your mouse and keyboard while SSH will only allow you to use the console/terminal.

If you are ssh-ing into a raspberry pi or vps remember that all the commands will be in linux. Click here for a quick introduction.

How to use SSH

The first step is to allow SSH on the device you are trying to connect to. Raspberry pi's natively support SSH so we will use it on there. In terminal enter the command [email protected]:~ $ sudo raspi-config You should be taken to a screen like this

Use the arrow keys to navigate to option #5 Interfacing Options, then select SSH and click "yes". If you aren't sure how to do this click here

SSH on mac/linux

If you're using macOS/linux SSH is natively built into your machine simply open your terminal and type ssh [email protected] with your raspberry pi's ip substituted for DEVICE.IP.HERE you can also replace pi with another account you have on your pi as long as you know the password. If you aren't sure how to do this please refer back to this tutorial.

SSH on Windows

Unlike mac or linux windows doesn't nativly support SSH so to use it you will have to download puTTY, a program to allow you to SSH from windows.

After you have successfully installed puTTY open it by typing putty in your search bar just make sure you are opening puTTY, not puTTYgen. (FYI: you can use puTTYgen to create key pairs to secure your device later)

Once puTTY is opened simply enter your raspberry pi's ip into the Host Name bar (mine is 192.168.1.32 ). Then click "Open".

Click yes/accept if you recieve any prompts. You will reach a screen asking you for credentials so log in with an account on your raspberry pi, again if you haven't done this please click here.

Similarities and Differences (TL;DR)

SSH and VNC both allow users to connect remotely to their devices allowing you to work from anywhere. Both use the local ip of the device you are trying to connect to in order to establish a connection (on a local network). Also multiple people can connect at the same time from anywhere.

They are different mainly because SSH is more restrictive, not allowing you to use the graphical desktop and only the terminal. Although for experienced programmers and those trying to make small changes this can be more convenient. Additionally with VNC, multiple users connecting at the same time will have to "battle" for control over the cursor and keyboard as it only takes one input at any given time. On the other hand, SSH allows multiple people to connect and do their own things with little if any interference from others on the system.
